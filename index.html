    데이터 직접 입력 거래·입출금·배당·분할 전체 관리

    입력 내역은 브라우저에 자동 저장되며(localStorage), Excel 업로드 데이터와 합산하여 계산됩니다.

    예: 1주가 10주로 분할되면 비율 10, 50주가 1주로 병합(역분할)되면 0.02. 분할일 이전 거래의 수량·단가가 자동 조정됩니다.

    데이터 관리 백업 · 복원 · 초기화

    수동 입력 거래·입출금, 카테고리, 목표 비중, GitHub 주소가 브라우저에 저장됩니다. 기기를 바꾸거나 데이터가 사라질 위험에 대비해 백업해 두실 수 있습니다. (Excel 업로드 데이터는 원본 파일로 다시 올리면 되므로 백업에는 포함되지 않습니다.)

    계산 방법 · 평균단가는 이동평균법(매수 수수료 원가 가산, 매도 수수료 매각대금 차감). 해외 종목의 원화 원가는 USD 원가 가중평균 매수환율로 환산. 해외 평가손익 분해: 주가요인 = 수량×(현재가−평단)×평균환율, 환율요인 = 수량×평단×(현재환율−평균환율), 교차항 = 잔여. 현금은 입출금−매수대금+매도대금+배당으로 추정한 원화 환산 근사치. XIRR은 입출금과 현재 총평가액 기준 연환산 내부수익률(Act/365). 분할이벤트 시트의 분할일 이전 거래는 수량×N·단가÷N으로 자동 조정. 미국 거래의 체결환율 공란은 환율히스토리로 보충.
    데이터 · 모든 계산은 이 파일을 연 브라우저 안에서만 수행되며 외부로 전송되지 않습니다. 카테고리·수동 입력 내역은 브라우저 localStorage에만 저장됩니다.
